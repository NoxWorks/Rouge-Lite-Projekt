# roguelite-project
Ein Dungeon Crawler im Stile von Spielen wie The Binding of Isaac oder Enter the Gungeon. Es wird von mir allein in der Godot Game Engine Entwickelt und nutzt die eingebaute Sprache der engine GDscript 

## Features
- Prozeduraler Dungeon-Generator
- Spielerbewegung und Kollision
- Erste Gegner-Logik
- Eigene Pixel-Art-Assets

## Technologien
- Godot Engine
- GDScript
- C# (Wenn nötig für einzelne Code segmente)
